# Spotify Clone App 🎵

A full-stack music manager using **Python**, **MySQL**, and **Streamlit**.

## Features

- 🔍 Search for songs by title, artist, or genre
- 🎵 Add new songs
- 📋 View all songs in a table
- ❌ Delete songs
- 📝 Edit song details

## Technologies Used

- 💻 Python
- 🧮 MySQL
- 🌐 Streamlit
